# Ts3Client

一个基于Android的TeamSpeak 3客户端应用

## 功能特性

- 连接到TeamSpeak 3服务器
- 语音通信
- 频道管理
- 服务器管理
- 主题切换（浅色/深色/动态）
- 权限管理

## 技术栈

- Kotlin
- Jetpack Compose (UI框架)
- Hilt (依赖注入)
- Room (本地存储)
- ts3j (TeamSpeak 3 Java库)

## 构建说明

1. 克隆仓库
2. 使用Android Studio打开项目
3. 配置签名信息
4. 构建并运行

## 依赖项

- AndroidX Core
- Jetpack Compose
- Room
- Hilt
- ts3j
- Gson
- Coil
- XXPermissions
